def check_data_freshness():
    """检查数据是否最新"""
    cursor = db.execute("SELECT MAX(stat_date) FROM ads_daily_summary")
    latest = cursor.fetchone()[0]

    if latest < (datetime.now() - timedelta(days=1)).strftime('%Y-%m-%d'):
        send_alert(f"数据不是最新的: {latest}")

## 🆘 故障恢复

### 场景1: 数据库损坏

```bash
# 从备份恢复
gunzip -k backups/newapi_warehouse_YYYYMMDD.db.gz
mv newapi_warehouse.db newapi_warehouse_broken.db
mv backups/newapi_warehouse_YYYYMMDD.db newapi_warehouse.db

# 重新更新
python3 daily_update.py
```

### 场景2: 服务无法访问

```bash
# 检查Nginx
sudo systemctl status nginx
sudo nginx -t
sudo systemctl restart nginx

# 检查Docker
docker-compose ps
docker-compose restart

# 检查端口
netstat -tlnp | grep :80
```

### 场景3: 数据更新失败

```bash
# 查看日志
tail -100 logs/daily_update.log

# 手动执行测试
python3 daily_update.py

# 检查数据库连接
sqlite3 newapi_warehouse.db ".tables"
```
