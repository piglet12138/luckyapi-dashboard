## 八、ADS层使用示例

### 8.1 核心业务指标查询

```sql
-- 最近30天核心指标
SELECT
    stat_date,
    active_users as dau,
    total_calls,
    total_revenue,
    arpu,
    arppu,
    conversion_rate
FROM ads_daily_summary
ORDER BY stat_date DESC
LIMIT 30;
```

### 8.2 增长趋势分析

```sql
-- 用户增长趋势
SELECT
    stat_date,
    registered_users,
    active_users,
    paying_users,
    activation_rate,
    conversion_rate
FROM ads_funnel_daily
ORDER BY stat_date DESC
LIMIT 30;
```

### 8.3 用户价值分析

```sql
-- 用户分层价值对比
SELECT
    segment,
    SUM(user_count) as total_users,
    SUM(total_calls) as total_calls,
    SUM(total_revenue) as total_revenue,
    ROUND(SUM(total_revenue) / SUM(user_count), 2) as revenue_per_user
FROM ads_user_segment_daily
GROUP BY segment
ORDER BY segment;
```

### 8.4 渠道效果评估

```sql
-- 渠道ROI分析
SELECT
    first_model,
    SUM(new_user_count) as users,
    SUM(paying_user_count) as paying_users,
    SUM(total_revenue) as revenue,
    ROUND(AVG(conversion_rate), 2) as avg_conversion_rate,
    ROUND(SUM(total_revenue) / SUM(new_user_count), 2) as ltv
FROM ads_channel_attribution
GROUP BY first_model
ORDER BY revenue DESC
LIMIT 20;
```
