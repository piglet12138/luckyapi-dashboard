# Token 分销平台数据深度深度分析报告

基于初步的AARRR漏斗分析，我们进行了更深度的量化下钻。以下是进一步深挖得到的详细数据支撑及相应的落地策略。

## 一、 核心财务与变现指标 (Monetization & LTV)

首先，我们量化了目前平台产生的交易流水与用户生命周期价值能力。

| 指标 | 数据 | 说明 |
| :--- | :--- | :--- |
| **总营收 (Revenue)** | ¥ 5,338.00 | 成功充值订单汇总 |
| **活跃用户数 (Active Users)** | 521 | 发起过至少 1 次调用的去重用户数 |
| **付费用户数 (Paying Users)** | 76 | 成功充值的去重用户数 |
| **渗透率 (Conversion Rate)** | 14.59% | 注册并试用后，转化为付费用户的比例 (表现**极为优秀**) |
| **ARPU** | ¥ 10.25 | 单个活跃用户的平均贡献收入 |
| **ARPPU** | ¥ 70.24 | **单付费用户平均收益** |

**💡 洞察与建议：**
- 14.59% 的付费渗透率在一众工具类/AI分销网站中属于**非常顶尖的水平**（行业均值普遍在 2%-5%）。这说明网站提供的模型质量、速度和定价是非常被市场认可的。
- ARPPU 达到 ¥ 70.24，这意味着只要能获取到一个付费用户，你就能赚到近70元。通过这个指标，你可以开始倒推**获客成本 (CAC，Customer Acquisition Cost)**。如果你的单个获客成本低于 20元，那么这就是一个极度赚钱的模型，可以考虑在各大平台（小红书/知乎/B站）开启付费投放。
- 考虑到复购率达 28.9%（22/76），这批用户的 LTV (长期价值) 还会随着时间进一步提升，建议推出**季度卡/年度大额充值包**来锁定高净值人群。


## 二、 留存率衰减分析 (Cohort Analysis)

上一步发现 7日留存仅有 16.9%，这次我们生成了具体的 Cohort 分析表，按“周”去观察用户的衰减曲线。

**样本数据片段（不同生命周期的群组）：**
*注：第0周为首次调用的当前周。*

| 首次使用周次 | 第0周留存 | 第1周留存 | 第2周留存 | 第3周留存 | 第4周留存 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **1月中旬客群** | 100% | 89.4% | 63.1% | 63.1% | 52.6% |
| **1月底客群** | 100% | 50.0% | 42.8% | 21.4% | 42.8% |
| **2月中旬客群** | 100% | 16.6% | 20.0% | 13.3% | 0.0% |

**💡 洞察与建议：**
这个表格揭示了一个非常致命的变化——**新发用户的留存质量正在急剧恶化！**
- 1月份进来的早期用户粘性极其可怕，第4周依然有高达 52% 的留存率。
- 但到了2月份中旬进来的客群，第1周留存直接跌到 **16.6%** 。
- 这可能是因为近期（2月底3月初）流量大涨，进来了过多“尝鲜”或“白嫖”流量。
- 必须立刻审视服务器响应延迟、由于滥用导致的账号配额耗尽等指标，检查近期是否有新用户体验受损的情况。

## 三、 寻找“Aha Moment”精确拐点

我们在第一步粗略地把阈值设为 5 次，但为了追求更完美的**Activation Metric**，我对用户按照更细的粒度进行了分箱：

| 调用次数边界 (Bins) | 总用户占比数 | 付费转化率 |
| :--- | :--- | :--- |
| **1 次就走** | 276 人 (绝大多数) | 0.00% |
| **2 次调用** | 32 人 | **46.88%** (首个高潮) |
| **3 - 10 次调用** | ~38 人 | < 10% |
| **11 - 20 次调用** | 17 人 | 23.53% |
| **21 - 50 次调用** | 31 人 | 29.03% |
| **超 100+ 次** | 106 人 (死忠粉) | **37.74%** |

**💡 洞察与建议：**
这里的数据出现了一个反直觉的“波峰波谷”现象，揭示了非常有意思的用户画像：
1. **即刻付费型 (波峰1)：** 触发了 2 次调用的这批人中，转化率惊人地高达46%。这说明有相当多人在“试了一下”之后立刻决定了充值。
2. **犹豫白嫖型 (波谷)：** 停留在 3-10 次的人转化率极低，多是以消耗赠送额度为主。
3. **高频使用者 (波峰2)：** 只要调用次数突破了大概 **20 次**，转化率就稳定保持在 30% 以上。

**行动方针**：设计**梯段性发力点**。
- **策略A**：为了抓住“2次即付费”的用户，在他们第1-2次使用时弹窗或者推送一个极低价格的“首充破冰专享包 (例如 0.99元/2元)”，让他们快速绑定支付手段。
- **策略B**：新手的 Aha Moment 应该定为 **20次**，设计类似于通关任务（如“完成您的第20次生成，解锁高级模型X的试用一天”）。


## 四、 产品热点：钱究竟是谁赚的？

最后我们深挖了免费用户与付费用户使用的底层大模型差异，它直接决定了以后的运营推广要打什么旗号。

**付费用户的Top3消耗模型：**
1. `gemini-3-pro-preview-thinking`
2. `claude-sonnet-4-6`
3. `gemini-3.1-pro-preview`

**免费（白嫖）用户的Top3消耗模型：**
1. `claude-haiku-4-5-20251001`
2. `gcp/gemini-3-pro-preview-thinking`
3. `claude-opus-4-6`

**💡 洞察与建议：**
- **付费用户偏好最强推理能力**。前三大模型（Gemini 3 Pro thinking 和 Claude 3.5 Sonnet）均为目前全球最抢手的推理和代码模型。说明付费用户多为了编程、深度思考买单。
- **白嫖用户偏好速问速答**。免费调用最多的 `Haiku` 属于快而便宜的模型。
- **营销建议**：网站对外的标语/推广海报，应该直接打出：**“低延迟、稳定支持 Gemini 3 Pro Thinking、Claude 3.5 Sonnet”**，甚至可以在主推界面把这两个模型挂在最醒目的位置。这是愿意掏钱那批人的刚需！
